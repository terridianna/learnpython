Variables
A variable is a word that refers to a value.
In Python values are integers, floats, and strings.

Integers are whole numbers like 1, 2, 3, -1, 100, 2000 and so on.
Floats are ratios like 1.5 or 0.25, a great example is money.
Strings are sentences, or a sequence of letters, numbers and symbols between double-quotes " " or single-quotes ' '.

Run the code in the sandbox.
